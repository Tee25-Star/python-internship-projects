# E-Commerce Analytical Dashboard

A comprehensive analytics dashboard for e-commerce data visualization built with Python and Streamlit.

## Features

- **Key Performance Indicators (KPIs)**
  - Total Revenue
  - Total Orders
  - Average Order Value
  - Unique Customers

- **Revenue Analytics**
  - Daily revenue trends
  - Monthly revenue breakdown
  - Revenue by day of week
  - Revenue by hour of day

- **Product Analysis**
  - Top products by revenue
  - Revenue distribution by category
  - Product performance metrics

- **Customer Analytics**
  - Customer lifetime value
  - Orders per customer distribution
  - Top customers analysis

- **Geographic Analysis**
  - Revenue by region
  - Order distribution by region

- **Interactive Filters**
  - Date range selection
  - Category filtering
  - Region filtering

- **Data Export**
  - Download summary data as CSV

## Installation

1. Install the required packages:
```bash
pip install -r requirements.txt
```

## Usage

Run the dashboard using Streamlit:

```bash
streamlit run app.py
```

Or double-click `run_dashboard.bat` for Windows.

The dashboard will open in your default web browser at `http://localhost:8501`

## Data

The dashboard uses synthetic e-commerce data generated by the `data_generator.py` module. The data includes:
- Order transactions
- Product information
- Customer data
- Geographic information
- Payment methods
- Order statuses

## Technologies Used

- **Streamlit**: Web framework for building the dashboard
- **Pandas**: Data manipulation and analysis
- **Plotly**: Interactive data visualization
- **NumPy**: Numerical computing

## Customization

You can customize the dashboard by:
- Modifying the data generator to use your own data source
- Adding new visualizations
- Adjusting the color schemes
- Adding new filters or metrics
