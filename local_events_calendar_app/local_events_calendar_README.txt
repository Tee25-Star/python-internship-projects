Local Events Calendar (Python)
==============================

What you get
------------
- A modern-looking desktop "Local Events Calendar" app (offline).
- Add / Edit / Delete events
- Search + filter by date range + category
- Autosaves to a JSON file
- Export to .ics (import into Google Calendar / Outlook)

Files
-----
- local_events_calendar.py
- local_events_calendar_events.json  (created automatically after first run)

How to run (Windows)
--------------------
1) Open PowerShell in this folder.
2) Run:

   python local_events_calendar.py

If "python" is not recognized, try:
   py local_events_calendar.py

Notes
-----
- Dates must be YYYY-MM-DD (example: 2026-01-28)
- Time must be HH:MM in 24-hour format (example: 18:30)
