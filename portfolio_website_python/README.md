# Python Portfolio Website (Flask)

A modern personal developer portfolio website with an animated, standout interface (glassmorphism cards + animated gradient blobs + smooth interactions).

## Run it

In PowerShell:

```powershell
cd "C:\Users\TREASURE.DESKTOP-L3QJAEP\portfolio_website_python"
python -m pip install -r requirements.txt
python app.py
```

Then open:

- `http://127.0.0.1:5000`

## Customize

- Edit your info in `data/profile.json`
- Edit your projects in `data/projects.json`

## Contact form

Submitting the form writes to:

- `data/messages.json`
