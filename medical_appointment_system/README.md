# MediConnect - Medical Appointment Booking System

MediConnect is a modern desktop application built with Python and Tkinter for managing **medical appointments**. It features a dark, card-based UI and a gradient-style header to make the interface stand out visually.

## Features

- **Book appointments**
  - Patient details: name, age, gender, contact number
  - Select doctor, date, time (24-hour format), and visit type
  - Add symptoms / notes
  - Prevents double-booking the same doctor at the same date and time

- **Manage appointments**
  - View all appointments in a table
  - Filter by patient or doctor name
  - Mark appointments as **Completed**
  - Cancel appointments

- **Dashboard stats**
  - Total appointments
  - Appointments scheduled **today**
  - **Upcoming** scheduled appointments

## Requirements

- Python 3.8+  
- Tkinter (comes bundled with standard Python on Windows)

No external packages are required.
